# ハードウェア仕様書

## 技術スタック (ハードウェア)

- **マイコンボード**: Raspberry Pi Pico W
- **モータードライバ**: MakerDrive (Cytron)
- **センサー**: 8ch フォトリフレクタセンサーアレイ
- **電源**: モバイルバッテリー または 単3電池 x 4

## ピン配置詳細

### モーター制御 (MakerDrive)

| デバイス | 機能 | ピン番号 | 備考 |
| --- | --- | --- | --- |
| **左モーター (M1)** | 前進 (Fwd) | GP5 | PWM制御 |
| | 後退 (Rev) | GP4 | PWM制御 |
| **右モーター (M2)** | 前進 (Fwd) | GP3 | PWM制御 |
| | 後退 (Rev) | GP2 | PWM制御 |

### ラインセンサー (8ch フォトリフレクタ)

左から右へ順番に配置されています。

| センサー番号 | ピン番号 | 重み (Weight) | 役割 |
| --- | --- | --- | --- |
| Sensor 0 | GP16 | -7 | 左端検出 (急旋回用) |
| Sensor 1 | GP17 | -5 | 左寄り検出 |
| Sensor 2 | GP18 | -3 | 左微調整 |
| Sensor 3 | GP19 | -1 | 中央左寄り |
| Sensor 4 | GP20 | +1 | 中央右寄り |
| Sensor 5 | GP21 | +3 | 右微調整 |
| Sensor 6 | GP22 | +5 | 右寄り検出 |
| Sensor 7 | GP28 | +7 | 右端検出 (急旋回用) |

### その他

| デバイス | 機能 | ピン番号 | 備考 |
| --- | --- | --- | --- |
