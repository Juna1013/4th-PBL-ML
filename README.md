# 4th-PBL - Line Trace Car Project

## 概要

Raspberry Pi Pico W を使用したライントレースカーの制御プログラムです。
8個のフォトリフレクタセンサーによる高精度なライントレース機能と、WiFi経由でのサーバー連携（リモート制御・音声コマンド）を兼ね備えています。

## 特徴

- **デュアルモード**: WiFi制御モードと、自律走行（スタンドアロン）モードを搭載。
- **高精度制御**: 8chセンサーとP制御により、滑らかなライン追従を実現。
- **拡張性**: REST APIを通じた外部システム（Webアプリ、音声認識など）との連携が可能。
- **モーター速度補正**: 左右のモーター個体差を補正する機能を実装。

## クイックスタート

### テストプログラムの実行

1. **センサーテスト**: 
   ```bash
   # test/unit_test/sensor_test.py を Pico W にアップロード
   # センサーが黒=0、白=1を正しく出力するか確認
   ```

2. **ライントレーステスト**:
   ```bash
   # test/integration_test/test.py を Pico W にアップロード
   # 8個のセンサーでライントレース動作を確認
   # 0.5秒ごとにセンサー状態が表示されます
   ```

### 本番環境の実行

1. **インストール**: `pico-w/` 内のファイルを Pico W に転送します。
2. **設定**: `main.py` 内の `WIFI_SSID`, `WIFI_PASSWORD` を編集します。
3. **実行**:
   - **通常モード**: `python main.py`
   - **ライントレースモード**: `python main.py standalone`
